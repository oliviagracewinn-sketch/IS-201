# IS AI Web Project

A small portfolio and scratch web project built for the final assignment.

## Files
- `index.html` — Professional homepage with résumé content and skills.
- `scratch.html` — From-scratch page with nested lists, local image, embedded YouTube video, live Tableau visualization, and on-page anchor.
- `app.html` — Single-page joke generator app with inline styles and script.
- `styles.css` — Shared professional stylesheet.
- `scratch.css` — Custom stylesheet for the scratch page.
- `scratch-planet.svg` — Local vector image used on the scratch page.

## Hosting with GitHub Pages
1. Create a GitHub repository and push these files to the repository root.
2. In the repository settings, enable **GitHub Pages** from the `main` branch and root folder.
3. Visit the deployed URL shown by GitHub Pages.

## Notes
- The scratch page is intentionally created from a blank HTML file and includes the required interactive content.
- `app.html` is linked from `scratch.html` and includes all app code on a single page.
- The embedded Tableau graph is live and interactive from Tableau Public.
